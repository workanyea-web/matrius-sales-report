# Matrius Sales Report Bot

Мультиагентная система ежедневного отчёта по воронке продаж для онлайн-школы Matrius.

**Реальная задача:** руководитель отдела продаж каждый день вручную собирает данные из amoCRM, считает конверсию и пишет отчёт в Telegram-чат команды. Этот проект автоматизирует весь процесс.

---

## Что делает система

1. Принимает данные о лидах за день (ввод или JSON-файл)
2. Запускает субагента-аналитика — считает конверсию, находит слабые этапы
3. Запускает субагента-редактора — форматирует Telegram-сообщение
4. Возвращает готовый текст, который можно скопировать и отправить

**Пример результата:**

```
📊 *Отчёт по продажам — 01.06.2026*

*Воронка за день:*
👤 Новые лиды: 14
📞 Дозвонились: 11 (79%)
📅 Записались на урок: 8
✅ Провели урок: 7 (88%)
💳 Оплатили: 3 (43%)

🏆 Топ дня: Мария (2 оплаты)

🔴 Конверсия в оплату 43% — ниже нормы (50%)
🔴 Конверсия в контакт 79% — ниже нормы (80%)

💡 *На завтра:* Разобрать 4 клиентов, которые провели урок но не оплатили — уточнить возражения и назначить дожим

⚠️ Конверсия в оплату ниже нормы — нужен разбор
```

---

## Архитектура

```
Пользователь вводит данные
         ↓
/sales-report (скилл-оркестратор)
         ↓
  @data-analyst (субагент)
  — считает конверсию по переходам
  — определяет топ-менеджера
  — выявляет аномалии
  — возвращает JSON
         ↓
  @report-writer (субагент)
  — читает JSON
  — форматирует Telegram-сообщение
         ↓
Готовый текст отчёта
```

**Модель:** последовательная + ролевая

## Как запустить

1. Открой папку проекта в Claude Code
2. Введи `/sales-report`
3. Отвечай на вопросы агента (или передай `example_input.json`)
4. Получи готовый текст отчёта

## Нормативы конверсии Matrius

| Этап | Норма |
|------|-------|
| Дозвон | ≥ 80% |

| Явка на урок | ≥ 80% |
